# Typ gry: Fantasy RPG (inspirowane Warcraftem)




## Krótkie:
* Wyciąganie broni (miecz wyciągany z pochwy, zrobienie kilku wersji dla różnych rodzajów broni)
* Obijające się stalowe bronie (obijające się dwa stalowe pręty, dodana głębia)
* Kroki (nagranie kroków w różnym rodzaju obuwia, podbicie)
* Dźwięk szaty maga (nagranie dźwięku materiału)
* Dźwięk zbroi (dźwięk obijającego się metalu, nisko)
* Erupcja magiczna (wybuch czegokolwiek, obniżony i zwolniony)
* Zapalany ogień/wybuch (dźwięk zapalania kuchenki gazowej, wycięty i obniżony)
* Odgłosy otrzymywania obrażeń (własny głos, zmodulowany do różnego typu postaci)
* Dźwięk wchodzenia do sklepu (upadające monety + pogłos)

## Średnio-długie:
* Głos demona (mój własny głos, obniżenie go, dodanie echa, zakłócenia)
* Dźwięk magicznego przedmiotu (dźwięk obijającego się metalu, przedłużony i podwyższony, kojarzy się z błyszczeniem, coś jak korzeń nirnu z Skyrim’a)
* Dźwięk rzucanego zaklęcia (światło) (dźwięk rzucanego materiału + metaliczny podźwięk)
* Dźwięk rzucanego zaklęcia (mrok) (dźwięk rzucanego materiału + metaliczny podźwięk, niższe tony)
* Pękająca ziemia (uderzanie pięścią w stół + łamanie/kruszenie czegoś + spadające kamienie)
* Jadące konie (obijanie palcami o stół)

## Długie:
* Castowanie potężnego zaklęcia (wiatr + metaliczne dźwięki, niskie tony, dudnienie, drżąca ziemia)
* Soundtrack (gitara akustyczna/elektryczna + gitara basowa + flet irlandzki)
* Odgłosy otoczenia (wiatr + szum drzew + odgłosy ludzi, zapętlone)
